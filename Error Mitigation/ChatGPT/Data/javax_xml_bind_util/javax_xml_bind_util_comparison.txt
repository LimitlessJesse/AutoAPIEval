Sample: 2 ValidationEventCollector
----------------------------------------------------------------
1. `void add(javax.xml.bind.ValidationEvent event)`: This method adds a ValidationEvent to the collector.
2. `List<javax.xml.bind.ValidationEvent> getEvents()`: This method returns a list of all ValidationEvents collected.
3. `int getEventCount()`: This method returns the total number of ValidationEvents collected.
4. `void reset()`: This method clears all collected ValidationEvents from the collector.
5. `boolean hasEvents()`: This method returns true if there are ValidationEvents collected, false otherwise.
