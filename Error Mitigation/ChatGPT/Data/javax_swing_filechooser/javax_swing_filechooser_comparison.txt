Sample: 0 FileFilter
----------------------------------------------------------------
1. `String getDescription()`: This method returns the description of the file filter.
2. `boolean accept(File f)`: This method determines whether the given file is accepted by the file filter.
3. `String getExtension()`: This method returns the extension used by the file filter.
4. `void setDescription(String description)`: This method sets the description for the file filter.
5. `void setExtensionListInDescription(boolean b)`: This method specifies whether the file extension should be shown in the description.
================================================================

Sample: 1 FileNameExtensionFilter
----------------------------------------------------------------
1. `String getDescription()`: This method returns the description of the file filter.
2. `String[] getExtensions()`: This method returns the array of file extensions that this filter accepts.
3. `boolean accept(File f)`: This method determines whether the given file is accepted by this filter.
4. `void setDescription(String description)`: This method sets the description for the file filter.
5. `void setExtensions(String[] extensions)`: This method sets the file extensions that this filter should accept.
================================================================

Sample: 2 FileSystemView
----------------------------------------------------------------
1. `File createNewFolder(File containingDir)`: This method creates a new folder with a default name in the specified directory.
2. `File getDefaultDirectory()`: This method returns the default directory to use for the file chooser.
3. `File getHomeDirectory()`: This method returns the user's home directory.
4. `File[] getRoots()`: This method returns the root directories available for the file system.
5. `File[] getFiles(File dir, boolean useFileHiding)` : This method returns the files in the specified directory, optionally including hidden files based on the useFileHiding parameter.
================================================================

Sample: 3 FileView
----------------------------------------------------------------
1. `String getDescription(File f)`: This method returns a human-readable description of the file.
2. `Icon getIcon(File f)`: This method returns an icon for the file.
3. `String getName(File f)`: This method returns the name of the file.
4. `String getTypeDescription(File f)`: This method returns a human-readable string representing the type of the file.
5. `Boolean isTraversable(File f)`: This method returns whether or not the file is a directory that can be traversed.
================================================================
