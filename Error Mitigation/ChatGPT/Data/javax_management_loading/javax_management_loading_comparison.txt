Sample: 0 DefaultLoaderRepository
----------------------------------------------------------------
1. `ClassLoader loadClass(String className)`: This method loads the class with the specified name.
2. `void addClassLoader(ClassLoader cl)`: This method adds the specified ClassLoader to the repository.
3. `void removeClassLoader(ClassLoader cl)`: This method removes the specified ClassLoader from the repository.
4. `ClassLoader getClassLoader(String className)`: This method returns the ClassLoader for the specified class name.
5. `int getClassLoaderCount()`: This method returns the number of ClassLoaders in the repository.
================================================================

Sample: 1 MLet
----------------------------------------------------------------
1. `void addURL(URL url)`: This method adds the specified URL to the list of URLs from which the MLet will load classes.
2. `void removeURL(URL url)`: This method removes the specified URL from the list of URLs from which the MLet will load classes.
3. `void addURLs(URL[] urls)`: This method adds an array of URLs to the list of URLs from which the MLet will load classes.
4. `void removeURLs(URL[] urls)`: This method removes an array of URLs from the list of URLs from which the MLet will load classes.
5. `void setLibraryDirectory(String libdir)`: This method sets the directory in which the MLet will look for libraries.
================================================================

Sample: 2 MLetContent
----------------------------------------------------------------
1. `void addMBean(String className)`: This method adds an MBean specified by its class name to the MLet content.
2. `void removeMBean(String className)`: This method removes an MBean specified by its class name from the MLet content.
3. `List<String> getMBeans()`: This method returns a list of class names of the MBeans in the MLet content.
4. `void setMBeans(List<String> mbeans)`: This method sets the list of class names of the MBeans in the MLet content.
5. `String getMBeanCount()`: This method returns the number of MBeans in the MLet content as a string.
================================================================

Sample: 3 PrivateMLet
----------------------------------------------------------------
1. `void addURL(URL url)`: Adds the specified URL to the list of URLs to search for classes and resources.
2. `void close()`: Closes this MLet.
3. `void removeURL(URL url)`: Removes the specified URL from the list of URLs to search for classes and resources.
4. `void setLibraryDirectory(String libdir)`: Sets the directory where libraries are stored.
5. `void setURLs(URL[] urls)`: Sets the list of URLs from which to load classes and resources.
================================================================
