Sample: 0 ChoiceCallback
----------------------------------------------------------------
1. `int getSelectedIndex()`: This method returns the index of the choice that was selected.
2. `String[] getChoices()`: This method returns the array of choices that were displayed.
3. `void setSelectedIndex(int selectedIndex)`: This method sets the index of the choice that should be selected.
4. `ChoiceCallback(String prompt, String[] choices, int defaultChoice, boolean multipleMode)`: Constructor that initializes the ChoiceCallback with the prompt, choices, default choice index, and multiple selection mode.
5. `boolean allowMultipleSelections()`: This method returns true if multiple choices can be selected.
================================================================

Sample: 1 ConfirmationCallback
----------------------------------------------------------------
1. `int getSelectedIndex()`: This method returns the index of the option that was selected by the user.
2. `String getSelectedValue()`: This method returns the value of the option that was selected by the user.
3. `void setSelectedIndex(int index)`: This method sets the index of the option that should be pre-selected.
4. `void setSelectedValue(String value)`: This method sets the value of the option that should be pre-selected.
5. `void setOptions(String[] options)`: This method sets the options that should be displayed to the user.
================================================================

Sample: 2 LanguageCallback
----------------------------------------------------------------
1. `Locale getLocale()`: This method returns the Locale used for localizing text.
2. `void setLocale(Locale locale)`: This method sets the Locale to be used for localizing text.
3. `String getLocaleString()`: This method returns the string representation of the Locale used for localizing text.
4. `void setLocaleString(String localeString)`: This method sets the Locale from a string representation for localizing text.
5. `void clear()`: This method clears the Locale information.
================================================================

Sample: 3 NameCallback
----------------------------------------------------------------
1. `String getName()`: This method returns the name associated with this NameCallback.
2. `void setName(String name)`: This method sets the name to be used for authentication.
3. `void clearName()`: This method clears the name.
4. `void setDefaultName(String defaultName)`: This method sets the default name to be used for authentication.
5. `String getDefaultName()`: This method returns the default name to be used for authentication.
================================================================

Sample: 4 PasswordCallback
----------------------------------------------------------------
1. `char[] getPassword()`: This method returns the password as a character array.
2. `void clearPassword()`: This method clears the password.
3. `void setPassword(char[] password)`: This method sets the password using the provided character array.
4. `String getPrompt()`: This method returns the prompt used to request the password.
5. `boolean isEchoOn()`: This method returns a boolean indicating whether the password should be displayed as it is being typed.
================================================================

Sample: 5 TextInputCallback
----------------------------------------------------------------
1. `String getText()`: This method returns the text information stored in the TextInputCallback.
2. `void setText(String text)`: This method sets the text information in the TextInputCallback to the specified value.
3. `boolean isEchoOn()`: This method returns a boolean indicating whether the text input should be echoed.
4. `void setEchoOn(boolean echoOn)`: This method sets whether the text input should be echoed.
5. `void clearText()`: This method clears the text information stored in the TextInputCallback.
================================================================

Sample: 6 TextOutputCallback
----------------------------------------------------------------
1. `void setText(String text)`: This method sets the text to be displayed.
2. `String getMessage()`: This method returns the message to be displayed.
3. `int getMessageType()`: This method returns the type of the message to be displayed.
4. `void setMessageType(int messageType)`: This method sets the type of the message to be displayed.
5. `void clear()`: This method clears the message to be displayed.
================================================================
