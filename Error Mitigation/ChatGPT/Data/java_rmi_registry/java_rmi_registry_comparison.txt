Sample: 0 LocateRegistry
----------------------------------------------------------------
1. `Registry createRegistry(int port)`: This method creates and exports a Registry on the specified port.
2. `Registry getRegistry()`: This method returns a reference to the default Registry on the local host.
3. `Registry getRegistry(String host, int port)`: This method returns a reference to the Registry on the specified host and port.
4. `void bind(String name, Remote obj)`: This method binds a remote reference to the specified name in this registry.
5. `void unbind(String name)`: This method removes the binding for the specified name in this registry.
