Sample: 0 OutputKeys
----------------------------------------------------------------
1. `String get(String key)`: This method retrieves the output property with the specified key.
2. `void set(String key, String value)`: This method sets the output property with the specified key to the specified value.
3. `Enumeration<String> keys()`: This method returns an enumeration of all the output keys.
4. `void clear()`: This method clears all the output properties.
5. `void remove(String key)`: This method removes the output property with the specified key.
================================================================

Sample: 1 Transformer
----------------------------------------------------------------
1. `void transform(Source xmlSource, Result outputTarget)`: This method transforms the XML Source tree into a Result tree.
2. `void setParameter(String name, Object value)`: This method sets a parameter for the transformation.
3. `void clearParameters()`: This method clears all parameters set for the transformation.
4. `void setErrorListener(ErrorListener listener)`: This method sets the error listener for the transformation.
5. `void setURIResolver(URIResolver resolver)`: This method sets the URI resolver for the transformation.
================================================================

Sample: 2 TransformerFactory
----------------------------------------------------------------
1. `Transformer newTransformer()`: This method creates a new Transformer object that performs a transformation from source to result.
2. `Templates newTemplates(Source source)`: This method creates a Templates object from the specified Source object.
3. `Transformer newTransformerHandler()`: This method creates a new TransformerHandler object that can be used to create a transformation result tree.
4. `Source getAssociatedStylesheet(Source source, String media, String title, String charset)`: This method gets the associated stylesheet for the specified source with the given media, title, and charset.
5. `void setFeature(String name, boolean value)`: This method sets the specified feature to the given value for this TransformerFactory instance.
================================================================
