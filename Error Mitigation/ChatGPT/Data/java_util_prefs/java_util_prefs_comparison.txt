Sample: 0 AbstractPreferences
----------------------------------------------------------------
1. `String[] childrenNames()`: This method returns an array of strings containing the names of the children of this node.
2. `void put(String key, String value)`: This method associates the specified value with the specified key in this preference node.
3. `String get(String key, String def)`: This method returns the value associated with the specified key in this preference node, or the specified default value if there is no value associated with the key.
4. `void remove(String key)`: This method removes the value associated with the specified key in this preference node, if any.
5. `void clear()`: This method removes all key-value pairs from this preference node.
================================================================

Sample: 1 NodeChangeEvent
----------------------------------------------------------------
1. `Preferences getChild()`: This method returns the child node that triggered the event.
2. `Preferences getParent()`: This method returns the parent node of the child node that triggered the event.
3. `String getKey()`: This method returns the key of the child node that triggered the event.
4. `boolean wasAdded()`: This method returns true if the child node was added, false if it was removed.
5. `boolean wasRemoved()`: This method returns true if the child node was removed, false if it was added.
================================================================

Sample: 2 PreferenceChangeEvent
----------------------------------------------------------------
1. `java.lang.String getKey()`: This method returns the key of the preference that was changed.
2. `java.util.prefs.Preferences getNode()`: This method returns the Preferences node that emitted the event.
3. `java.lang.String getNewValue()`: This method returns the new value of the preference that was changed.
4. `java.lang.String getOldValue()`: This method returns the old value of the preference that was changed.
5. `java.lang.String toString()`: This method returns a string representation of the PreferenceChangeEvent.
================================================================

Sample: 3 Preferences
----------------------------------------------------------------
1. `String get(String key, String def)`: This method retrieves the value associated with the specified key in this preference node, or returns the default value if the key is not found.
2. `void put(String key, String value)`: This method associates the specified value with the specified key in this preference node.
3. `String[] keys()`: This method returns an array of the keys in this preference node.
4. `void remove(String key)`: This method removes the value associated with the specified key in this preference node.
5. `Preferences node(String pathName)`: This method returns the preference node at the specified path name in this preference node.
