Sample: 0 AttributeList
----------------------------------------------------------------
1. `String getName()`: This method returns the name of the attribute.
2. `String getValue()`: This method returns the value of the attribute.
3. `AttributeList getNext()` : This method returns the next attribute in the list.
4. `boolean checkValue(String value)`: This method checks if the attribute value matches the specified value.
5. `boolean checkName(String name)`: This method checks if the attribute name matches the specified name.
================================================================

Sample: 1 ContentModel
----------------------------------------------------------------
1. `void addElement(ContentModel contentModel)`: This method adds a ContentModel to the current content model.
2. `void addElement(int index, ContentModel contentModel)`: This method inserts a ContentModel at the specified index in the content model.
3. `void removeElement(ContentModel contentModel)`: This method removes a ContentModel from the content model.
4. `ContentModel getElement(int index)`: This method returns the ContentModel at the specified index in the content model.
5. `int getSize()`: This method returns the number of ContentModels in the content model.
================================================================

4. `void parse(Reader in, HTMLEditorKit.ParserCallback callback, boolean ignoreCharSet, int pos, boolean last): This method parses the HTML document from the given Reader and invokes the specified ParserCallback, starting at the specified position and indicating if it is the last part of the document.
5. `void parse(Reader in, HTMLEditorKit.ParserCallback callback, int pos, int length, boolean last)`: This method parses a specific portion of the HTML document from the given Reader and invokes the specified ParserCallback, starting at the specified position, with a specified length, and indicating if it is the last part of the document.
