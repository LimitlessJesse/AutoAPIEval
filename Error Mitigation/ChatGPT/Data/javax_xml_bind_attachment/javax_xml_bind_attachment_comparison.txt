----------------------------------------------------------------

2. `byte[] getAttachmentAsByteArray(String contentId)`: This method returns the attachment identified by the specified content ID as a byte array.
3. `InputStream getAttachmentAsInputStream(String contentId)`: This method returns the attachment identified by the specified content ID as an InputStream.
4. `Source getAttachmentAsSource(String contentId)`: This method returns the attachment identified by the specified content ID as a javax.xml.transform.Source.
