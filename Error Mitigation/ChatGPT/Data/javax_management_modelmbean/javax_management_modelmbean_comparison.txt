Sample: 0 DescriptorSupport
----------------------------------------------------------------
1. `void setField(String fieldName, Object fieldValue)`: This method sets the value of the specified field in the descriptor.
2. `Object getField(String fieldName)`: This method returns the value of the specified field in the descriptor.
3. `void setDescriptorType(String type)`: This method sets the type of the descriptor.
4. `String getDescriptorType()`: This method returns the type of the descriptor.
5. `void removeField(String fieldName)`: This method removes the specified field from the descriptor.

Sample: 1 ModelMBeanAttributeInfo
----------------------------------------------------------------
1. `String getDescription()`: This method returns the description of the attribute.
2. `String getName()`: This method returns the name of the attribute.
3. `boolean isReadable()`: This method returns true if the attribute is readable, false otherwise.
4. `boolean isWritable()`: This method returns true if the attribute is writable, false otherwise.
5. `String getType()`: This method returns the type of the attribute.
================================================================
