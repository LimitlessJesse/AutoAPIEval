Sample: 0 ServerSocketFactory
----------------------------------------------------------------
1. `ServerSocket createServerSocket(int port)`: This method creates a server socket bound to the specified port.
2. `ServerSocket createServerSocket(int port, int backlog)`: This method creates a server socket bound to the specified port with a specific backlog.
3. `ServerSocket createServerSocket(int port, int backlog, InetAddress bindAddr)`: This method creates a server socket bound to the specified port with a specific backlog and bind address.
4. `ServerSocket createServerSocket()`: This method creates an unbound server socket.
5. `ServerSocket createServerSocket(int port, int backlog, InetAddress bindAddr, int timeout)`: This method creates a server socket bound to the specified port with a specific backlog, bind address, and timeout.

Sample: 1 SocketFactory
----------------------------------------------------------------
1. `Socket createSocket()`: This method creates a socket without specifying any parameters.
2. `Socket createSocket(String host, int port)`: This method creates a socket and connects it to the specified host and port.
3. `Socket createSocket(InetAddress address, int port)`: This method creates a socket and connects it to the specified InetAddress and port.
4. `Socket createSocket(String host, int port, InetAddress localAddress, int localPort)`: This method creates a socket and connects it to the specified host and port, using the specified local address and port.
5. `Socket createSocket(InetAddress address, int port, InetAddress localAddress, int localPort)`: This method creates a socket and connects it to the specified InetAddress and port, using the specified local address and port.
