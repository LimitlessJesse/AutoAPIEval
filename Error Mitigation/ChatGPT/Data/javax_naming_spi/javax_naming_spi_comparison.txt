Sample: 0 DirectoryManager
----------------------------------------------------------------
1. `Object getAttribute(DirContext ctx, Name name)`: This method retrieves the named attribute of a named object.
2. `Attributes getAttributes(DirContext ctx, Name name)`: This method retrieves the attributes associated with a named object.
3. `void modifyAttributes(DirContext ctx, Name name, ModificationItem[] mods)`: This method modifies the attributes associated with a named object using an ordered list of modifications.
4. `void bind(DirContext ctx, Name name, Object obj, Attributes attrs)`: This method creates a new named object, adding the attributes specified in the attributes parameter.
5. `void rebind(DirContext ctx, Name name, Object obj, Attributes attrs)`: This method replaces the attributes of a named object with the attributes specified in the attributes parameter.
================================================================

Sample: 1 DirStateFactory.Result
----------------------------------------------------------------
1. `Object getObject()`: This method returns the object associated with this DirStateFactory.Result.
2. `Attributes getAttributes()`: This method returns the attributes associated with this DirStateFactory.Result.
3. `NameParser getParser()`: This method returns the NameParser associated with this DirStateFactory.Result.
4. `Name getName()`: This method returns the name associated with this DirStateFactory.Result.
5. `Object getState()`: This method returns the state associated with this DirStateFactory.Result.
================================================================

Sample: 2 NamingManager
----------------------------------------------------------------
1. `Object getObjectInstance(Object obj, Name name, Context nameCtx, Hashtable<?, ?> environment)`: This method creates an object using the specified object and location information.
2. `Context getInitialContext(Hashtable<?, ?> environment)`: This method creates an initial context using the specified environment.
3. `Object getStateToBind(Object obj, Name name, Context nameCtx, Hashtable<?, ?> environment)`: This method returns the state to bind for the specified object and location information.
4. `Object getURLContext(String scheme, Hashtable<?, ?> environment)`: This method creates a context for the specified URL scheme and environment.
5. `Object getContinuationContext(CannotProceedException cpe)`: This method creates a continuation context for the specified CannotProceedException.
================================================================

Sample: 3 ResolveResult
----------------------------------------------------------------
1. `Object getResolvedObj()`: This method returns the resolved object.
2. `NameParser getParser()`: This method returns the name parser used for resolving the name.
3. `Name getName()`: This method returns the original unresolved name.
4. `Object getRemainingName()`: This method returns the remaining unresolved name.
5. `boolean isRelative()`: This method returns true if the resolved object is relative to the context in which the name was resolved.
================================================================
