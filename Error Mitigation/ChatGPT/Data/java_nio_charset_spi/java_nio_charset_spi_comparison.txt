Sample: 0 CharsetProvider
----------------------------------------------------------------
1. `Charset charsetForName(String charsetName)`: This method returns a charset object for the given charset name.
2. `Iterator<Charset> charsets()`: This method returns an iterator over the available charsets.
3. `boolean isSupported(String charsetName)`: This method checks if the given charset name is supported by this provider.
4. `String[] aliases(String charsetName)`: This method returns an array of aliases for the given charset name.
5. `CharsetDecoder newDecoder(Charset cs)`: This method creates a new charset decoder for the given charset.
