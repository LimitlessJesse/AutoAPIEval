Sample: 0 AbstractUndoableEdit
----------------------------------------------------------------
1. `void die()`: This method is called when this edit is no longer used.
2. `boolean canRedo()`: Returns true if this edit is alive and has been undone.
3. `boolean canUndo()`: Returns true if this edit is alive and has not been undone.
4. `void redo()`: Redoes the edit.
5. `void undo()`: Undoes the edit.
================================================================

Sample: 1 CompoundEdit
----------------------------------------------------------------
1. `void end()`: This method ends the edit sequence.
2. `void die()`: This method cleans up any resources used by this edit.
3. `boolean isInProgress()`: This method returns true if this edit is in progress.
4. `void undo()`: This method undoes the edit.
5. `void redo()`: This method redoes the edit.
================================================================

Sample: 2 StateEdit
----------------------------------------------------------------
1. `void end()`: This method ends the edit sequence.
2. `boolean canRedo()`: This method returns true if there are edits to be redone.
3. `boolean canUndo()`: This method returns true if there are edits to be undone.
4. `void die()`: This method discards the edit and releases any resources used by it.
5. `void undo()`: This method undoes the edit.
================================================================

Sample: 3 UndoableEditSupport
----------------------------------------------------------------
1. `void addUndoableEditListener(UndoableEditListener l)`: This method adds an UndoableEditListener to the list that's notified each time an UndoableEdit is added.
2. `void removeUndoableEditListener(UndoableEditListener l)`: This method removes an UndoableEditListener from the list.
3. `void postEdit(UndoableEdit e)`: This method posts an UndoableEdit to all the UndoableEditListeners.
4. `void end()`: This method ends the current compound edit and sends it to the listeners.
5. `boolean isInProgress()`: This method returns true if an UndoableEdit is in progress, false otherwise.
================================================================

Sample: 4 UndoManager
----------------------------------------------------------------
1. `void undo()`: This method undoes the last edit.
2. `void redo()`: This method redoes the last undone edit.
3. `boolean canUndo()`: This method returns true if edits can be undone.
4. `boolean canRedo()`: This method returns true if edits can be redone.
5. `void discardAllEdits()`: This method discards all edits in the manager.
================================================================
