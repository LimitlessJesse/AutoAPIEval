Sample: 0 Schema
----------------------------------------------------------------
1. `javax.xml.validation.Validator newValidator()`: This method returns a new instance of a validator for this schema.
2. `javax.xml.validation.ValidatorHandler newValidatorHandler()`: This method returns a new instance of a validator handler for this schema.
3. `javax.xml.validation.TypeInfoProvider getTypeInfoProvider()`: This method returns the type information provider for this schema.
4. `javax.xml.validation.ValidatorHandler newValidatorHandler(javax.xml.transform.Source source)`: This method returns a new instance of a validator handler for this schema using the specified source.
5. `javax.xml.validation.ValidatorHandler newValidatorHandler(javax.xml.transform.Source[] schemas)`: This method returns a new instance of a validator handler for this schema using the specified array of sources.

----------------------------------------------------------------

3. `javax.xml.validation.Schema newSchema(InputStream is)` : This method creates a new Schema object from an InputStream representing a schema.
4. `javax.xml.validation.Schema newSchema(Source[] schemas)` : This method creates a new Schema object from an array of schema Sources.
5. `javax.xml.validation.Schema newSchema(Source schema, LSResourceResolver resourceResolver)` : This method creates a new Schema object from a schema Source with a specified LSResourceResolver.

Sample: 3 TypeInfoProvider
----------------------------------------------------------------

3. `TypeInfo getAttributeTypeInfo(QName name)`: This method returns the type information of the attribute with the specified QName.

5. `TypeInfo getDeclaredAttributeTypeInfo(int index)`: This method returns the type information of the declared attribute at the specified index.
