Sample: 1 GSSManager
----------------------------------------------------------------
1. `GSSName createName(String name, Oid nameType)`: This method creates a GSSName object representing the given name and name type.
2. `GSSCredential createCredential(GSSName desiredName, int lifetime, Oid[] mechanism, int usage)`: This method creates a GSSCredential object with the specified parameters.
3. `GSSContext createContext(GSSName peer, Oid mech, GSSCredential myCred, int lifetime)`: This method creates a GSSContext object for communication with the specified peer using the specified mechanism and credentials.
4. `GSSContext getContext(byte[] interProcessToken)`: This method creates a GSSContext object from the provided inter-process token.
5. `Oid[] getMechs()`: This method returns an array of Oid objects representing the supported mechanisms.
