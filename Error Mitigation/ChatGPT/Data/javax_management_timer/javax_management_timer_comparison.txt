Sample: 0 Timer
----------------------------------------------------------------
1. `void start()`: This method starts the timer.
2. `void stop()`: This method stops the timer.
3. `long getNotificationCount()`: This method returns the number of notifications sent by the timer.
4. `void setPeriod(long period)`: This method sets the period of the timer.
5. `long getPeriod()`: This method returns the period of the timer.
================================================================

Sample: 1 TimerNotification
----------------------------------------------------------------
1. `String getNotificationType()`: This method returns the type of the notification.
2. `Object getUserData()`: This method returns the user data associated with the notification.
3. `long getTimeStamp()`: This method returns the timestamp of the notification.
4. `String getTimerName()`: This method returns the name of the timer that triggered the notification.
5. `String getTimerNotificationInfo()`: This method returns additional information about the timer notification.
