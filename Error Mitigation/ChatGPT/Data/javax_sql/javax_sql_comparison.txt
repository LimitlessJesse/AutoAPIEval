Sample: 0 ConnectionEvent
----------------------------------------------------------------
1. `ConnectionEvent(Object source)`: This constructor creates a ConnectionEvent object with the specified source.
2. `ConnectionEvent(Object source, int eventType)`: This constructor creates a ConnectionEvent object with the specified source and event type.
3. `int getEventType()`: This method returns the event type of the ConnectionEvent.
4. `Object getSource()`: This method returns the source of the ConnectionEvent.
5. `String toString()`: This method returns a string representation of the ConnectionEvent.
================================================================

Sample: 1 RowSetEvent
----------------------------------------------------------------
1. `RowSetEvent(Object source)`: This constructor creates a new RowSetEvent object with the specified source.
2. `RowSetEvent(Object source, int id)`: This constructor creates a new RowSetEvent object with the specified source and event ID.
3. `int getID()`: This method returns the ID of the event.
4. `RowSet getSource()`: This method returns the RowSet object that is the source of the event.
5. `String toString()`: This method returns a string representation of the RowSetEvent object.

Sample: 2 StatementEvent
----------------------------------------------------------------
1. `Statement getStatement()`: This method returns the Statement object that is the source of the event.
2. `PooledConnection getConnection()`: This method returns the PooledConnection object that created the Statement object.
3. `int getEventType()`: This method returns the type of the event.
4. `SQLException getSQLException()`: This method returns the SQLException that caused the event to be generated.
5. `String toString()`: This method returns a string representation of the StatementEvent.
