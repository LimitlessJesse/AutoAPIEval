Sample: 0 ZoneOffsetTransition
----------------------------------------------------------------
1. `ZoneOffset getOffsetBefore()`: Returns the offset before the transition.
2. `ZoneOffset getOffsetAfter()`: Returns the offset after the transition.
3. `boolean isGap()`: Returns true if the transition is a gap, false if it is an overlap.
4. `boolean isValidOffset(ZoneOffset offset)`: Checks if the specified offset is valid for this transition.
5. `Instant getInstant()`: Returns the transition instant.
================================================================

Sample: 1 ZoneOffsetTransitionRule
----------------------------------------------------------------
1. `ZoneOffsetTransition createTransition(LocalDateTime dateTime)`: This method creates a transition at the specified date and time.
2. `ZoneOffsetTransition createTransition(int year)`: This method creates a transition at the specified year.
3. `ZoneOffsetTransition createTransition(int year, Month month, int dayOfMonth, LocalTime time, boolean timeEndOfDay, ZoneOffsetTransitionRule.TimeDefinition timeDefnition)`: This method creates a transition at the specified year, month, day, time, and time definition.
4. `boolean isTransition(LocalDate localDate)`: This method checks if a transition occurs on the specified date.
5. `ZoneOffsetTransitionRule createRule(Month month, int dayOfMonthIndicator, DayOfWeek dayOfWeek, LocalTime time, boolean timeEndOfDay, ZoneOffsetTransitionRule.TimeDefinition timeDefinition, ZoneOffset standardOffset, ZoneOffset offsetBefore, ZoneOffset offsetAfter)`: This method creates a rule for a transition with the specified parameters.
================================================================

Sample: 2 ZoneRules
----------------------------------------------------------------
1. `ZoneOffset getOffset(LocalDateTime dateTime)`: This method returns the offset applicable at the specified local date-time.
2. `ZoneOffsetTransition nextTransition(Instant instant)`: This method returns the next transition after the specified instant.
3. `boolean isFixedOffset()`: This method returns true if the zone rules are fixed offset, false otherwise.
4. `List<ZoneOffsetTransition> getTransitions()`: This method returns a list of all the transitions in the zone rules.
5. `ZoneOffsetTransition getTransition(LocalDateTime dateTime)`: This method returns the transition applicable at the specified local date-time.
================================================================

Sample: 3 ZoneRulesProvider
----------------------------------------------------------------
1. `Set<String> getAvailableZoneIds()`: This method returns a set of all available zone IDs.
2. `ZoneRules getRules(String zoneId)`: This method returns the time-zone rules for the specified zone ID.
3. `ZoneOffset getOffset(Instant instant)`: This method returns the offset for the specified instant in the UTC time-zone.
4. `boolean isFixedOffset()`: This method returns true if the time-zone rules are fixed offset from UTC.
5. `ZoneOffsetTransitionRules getTransitionRules()`: This method returns the transition rules for the time-zone.
================================================================
