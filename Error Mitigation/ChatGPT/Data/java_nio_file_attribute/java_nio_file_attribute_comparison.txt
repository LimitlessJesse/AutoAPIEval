Sample: 0 AclEntry
----------------------------------------------------------------
1. `UserPrincipal principal()`: Returns the UserPrincipal associated with this ACL entry.
2. `Set<AclEntryPermission> permissions()`: Returns the set of permissions for this ACL entry.
3. `Set<AclEntryFlag> flags()`: Returns the set of flags for this ACL entry.
4. `AclEntryType type()`: Returns the type of this ACL entry.
5. `boolean isNegative()`: Returns true if this ACL entry represents a negative permission.
================================================================

Sample: 1 AclEntry.Builder
----------------------------------------------------------------
1. `AclEntry.Builder setPrincipal(UserPrincipal principal)`: This method sets the principal for the AclEntry.
2. `AclEntry.Builder setType(AclEntryType type)`: This method sets the type for the AclEntry.
3. `AclEntry.Builder setPermissions(Set<AclEntryPermission> permissions)`: This method sets the permissions for the AclEntry.
4. `AclEntry.Builder setFlags(Set<AclEntryFlag> flags)`: This method sets the flags for the AclEntry.
5. `AclEntry build()`: This method constructs a new AclEntry based on the current state of the builder.

3. `String getOwner(FileAttribute<?> attribute)`: This method returns the owner of the specified file attribute.

----------------------------------------------------------------
