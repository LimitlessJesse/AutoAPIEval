Sample: 0 Files
----------------------------------------------------------------
1. `boolean exists(Path path)`: This method checks if a file or directory exists at the specified path.
2. `void createDirectory(Path dir)`: This method creates a new directory at the specified path.
3. `InputStream newInputStream(Path path)`: This method opens a file for reading, returning an InputStream to read from the file.
4. `OutputStream newOutputStream(Path path)`: This method opens or creates a file for writing, returning an OutputStream to write to the file.
5. `List<String> readAllLines(Path path)`: This method reads all lines from a file and returns them as a List of Strings.
================================================================

Sample: 1 FileStore
----------------------------------------------------------------
1. `long getTotalSpace()`: Returns the total space in bytes for the file store.
2. `long getUsableSpace()`: Returns the number of bytes available to this file store.
3. `long getUnallocatedSpace()`: Returns the number of bytes of unallocated space in the file store.
4. `String type()`: Returns the type of the file store.
5. `boolean isReadOnly()`: Returns true if the file store is read-only.
