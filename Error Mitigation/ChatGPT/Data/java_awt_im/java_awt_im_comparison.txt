Sample: 0 InputContext
----------------------------------------------------------------
1. `void dispatchInputMethodEvent(InputMethodEvent event)`: This method dispatches an input method event to the active input method.
2. `void reconvert()` : This method requests re-conversion from the current input method.
3. `void setCharacterSubsets(Character.Subset[] subsets)`: This method sets the subsets of the Unicode character set that input methods should be allowed to input.
4. `void setCompositionEnabled(boolean enable)`: This method enables or disables the input method for composition, depending on the boolean parameter.
5. `void setLocale(Locale locale)`: This method sets the current input locale.
