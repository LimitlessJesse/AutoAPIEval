Sample: 0 PrintEvent
----------------------------------------------------------------
1. `PrintEvent(Object source)`: This constructor creates a new PrintEvent object with the specified source.
2. `PrintEvent(Object source, int reason)`: This constructor creates a new PrintEvent object with the specified source and reason.
3. `int getPrintEventType()`: This method returns the type of the print event.
4. `String toString()`: This method returns a string representation of the PrintEvent object.
5. `void setPrintEventType(int type)`: This method sets the type of the print event.

Sample: 1 PrintJobAdapter
----------------------------------------------------------------
1. `void printDataTransferCompleted(PrintJobEvent pje)`: This method is called when a print job has successfully transferred data to the printer.
2. `void printJobCompleted(PrintJobEvent pje)`: This method is called when a print job has successfully completed.
3. `void printJobFailed(PrintJobEvent pje)`: This method is called when a print job has failed to complete.
4. `void printJobCanceled(PrintJobEvent pje)`: This method is called when a print job has been canceled.
5. `void printJobNoMoreEvents(PrintJobEvent pje)`: This method is called when there are no more events for a print job.
