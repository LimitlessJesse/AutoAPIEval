Sample: 0 Book
----------------------------------------------------------------
1. `void append(PageFormat format, Printable painter)`: This method appends a new page to the end of the Book with the specified PageFormat and Printable.
2. `int getNumberOfPages()`: This method returns the total number of pages in the Book.
3. `PageFormat getPageFormat(int pageIndex)`: This method returns the PageFormat of the page at the specified index.
4. `Printable getPrintable(int pageIndex)`: This method returns the Printable object for the page at the specified index.
5. `void setPage(int pageIndex, Printable painter, PageFormat format)`: This method sets the Printable object and PageFormat for the page at the specified index.
================================================================

Sample: 1 PageFormat
----------------------------------------------------------------
1. `double getWidth()`: This method returns the width of the page in points.
2. `double getHeight()`: This method returns the height of the page in points.
3. `int getOrientation()`: This method returns the orientation of the page (either PORTRAIT or LANDSCAPE).
4. `void setOrientation(int orientation)`: This method sets the orientation of the page.
5. `Paper getPaper()`: This method returns the Paper object associated with the PageFormat.
================================================================

Sample: 2 Paper
----------------------------------------------------------------
1. `double getWidth()`: This method returns the width of the paper in points.
2. `double getHeight()`: This method returns the height of the paper in points.
3. `void setSize(double width, double height)`: This method sets the size of the paper to the specified width and height in points.
4. `void setImageableArea(double x, double y, double width, double height)`: This method sets the imageable area of the paper to the specified values in points.
5. `void setOrientation(int orientation)`: This method sets the orientation of the paper to the specified value.
================================================================

Sample: 3 PrinterJob
----------------------------------------------------------------
1. `void setPrintable(Printable painter)`: This method sets the Printable object to use for printing.
2. `void setPrintService(PrintService service)`: This method sets the PrintService to use for printing.
3. `void setPageable(Pageable document)`: This method sets the Pageable object to use for printing.
4. `void print()`: This method initiates a print job with the current settings.
5. `PageFormat pageDialog(PageFormat page)`: This method displays a page setup dialog and returns the chosen PageFormat.
================================================================
