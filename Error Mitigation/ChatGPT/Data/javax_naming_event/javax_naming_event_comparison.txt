Sample: 0 NamingEvent
----------------------------------------------------------------
1. `Object getSource()`: This method returns the object on which the event initially occurred.
2. `int getType()`: This method returns the type of the event.
3. `Object getNewBinding()`: This method returns the new binding that resulted from the event.
4. `Object getOldBinding()`: This method returns the old binding that was replaced by the event.
5. `String toString()`: This method returns a string representation of the event.
================================================================

Sample: 1 NamingExceptionEvent
----------------------------------------------------------------
1. `NamingException getException()`: This method returns the NamingException that caused the event to be fired.
2. `NamingEvent getNamingEvent()`: This method returns the NamingEvent associated with this NamingExceptionEvent.
3. `void dispatch(NamingListener listener)`: This method dispatches the event to the specified NamingListener.
4. `void dispatch(Object target, NamingListener listener)`: This method dispatches the event to the specified target and NamingListener.
5. `void dispatch(NamingListener[] listeners)`: This method dispatches the event to the array of NamingListeners.
================================================================
