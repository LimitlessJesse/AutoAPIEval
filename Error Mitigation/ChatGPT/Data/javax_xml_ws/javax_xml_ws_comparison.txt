Sample: 0 Endpoint
----------------------------------------------------------------
1. `void publish(Object implementor)`: This method publishes the endpoint at the given address with the provided implementor.
2. `void stop()`: This method stops the endpoint.
3. `void setExecutor(Executor executor)`: This method sets the executor for this endpoint.
4. `Binding getBinding()`: This method returns the binding for this endpoint.
5. `EndpointReference getEndpointReference(Element... referenceParameters)`: This method returns the endpoint reference for this endpoint with the specified reference parameters.
================================================================

Sample: 1 EndpointContext
----------------------------------------------------------------
1. `void addEndpoint(String name, Object implementor)`: This method adds an endpoint with the specified name and implementor.
2. `void removeEndpoint(String name)`: This method removes the endpoint with the specified name.
3. `Object getEndpoint(String name)`: This method returns the implementor object associated with the specified endpoint name.
4. `Set<String> getEndpointNames()`: This method returns a set of all endpoint names currently registered.
5. `boolean containsEndpoint(String name)`: This method checks if an endpoint with the specified name is registered.
================================================================

Sample: 2 EndpointReference
----------------------------------------------------------------
1. `String getAddress()`: This method returns the address of the endpoint.
2. `Map<QName, String> getElements()`: This method returns a map of the elements in the endpoint reference.
3. `List<Element> getReferenceParameters()`: This method returns a list of reference parameters in the endpoint reference.
4. `String getWSDLDocumentLocation()`: This method returns the location of the WSDL document for the endpoint.
5. `void setAddress(String address)`: This method sets the address of the endpoint.
================================================================

Sample: 3 Holder
----------------------------------------------------------------
1. `T value()`: This method returns the value held by the Holder.
2. `void value(T value)`: This method sets the value held by the Holder to the specified value.
3. `Holder<T> createAndSet(T value)`: This method creates a new Holder with the specified value and sets it.
4. `void clear()`: This method clears the value held by the Holder.
5. `boolean isNull()`: This method checks if the value held by the Holder is null.
================================================================

Sample: 4 RespectBindingFeature
----------------------------------------------------------------
1. `boolean isEnabled()`: This method returns true if the RespectBindingFeature is enabled, false otherwise.
2. `void setEnabled(boolean enabled)`: This method enables or disables the RespectBindingFeature.
3. `String getID()`: This method returns the unique identifier for the RespectBindingFeature.
4. `void setID(String id)`: This method sets the unique identifier for the RespectBindingFeature.
5. `boolean isFeatureEnabled()`: This method returns true if the RespectBindingFeature is enabled, false otherwise.
================================================================

Sample: 5 Service
----------------------------------------------------------------
1. `QName getServiceName()`: This method returns the qualified name of the service.
2. `URL getWSDLDocumentLocation()`: This method returns the location of the WSDL document for the service.
3. `EndpointReference getWSDLDocumentLocation( QName serviceName )`: This method returns the location of the WSDL document for the specified service name.
4. `Executor getExecutor()`: This method returns the executor for this service.
5. `HandlerResolver getHandlerResolver()`: This method returns the handler resolver for this service.
================================================================

Sample: 7 WebServicePermission
----------------------------------------------------------------
1. `String getName()`: This method returns the name of the web service permission.
2. `boolean implies(Permission permission)`: This method checks if the specified permission is implied by this web service permission.
3. `void checkGuard(Object object)`: This method checks if the specified object's permission is allowed.
4. `String getActions()`: This method returns the actions of the web service permission.
5. `PermissionCollection newPermissionCollection()`: This method creates a new permission collection for storing web service permissions.
================================================================
