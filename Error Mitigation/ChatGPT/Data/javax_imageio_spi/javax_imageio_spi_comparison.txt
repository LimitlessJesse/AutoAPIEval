Sample: 0 IIORegistry
----------------------------------------------------------------
1. `void registerServiceProvider(Object provider)`: This method registers a service provider object with this registry.
2. `void deregisterServiceProvider(Object provider)`: This method deregisters a service provider object from this registry.
3. `Iterator<ServiceRegistry.Filter> getServiceProviders(Class<?> providerClass, boolean returnInstances)`: This method returns an iterator over all the service provider instances of a given class.
4. `Iterator<ServiceRegistry.Filter> getServiceProviders(Class<?> providerClass, ServiceRegistry.Filter filter, boolean returnInstances)`: This method returns an iterator over all the service provider instances of a given class that match the specified filter.
5. `void setOrdering(Class<?> category, Object firstProvider, Object secondProvider)`: This method sets the ordering of two service provider objects within a given category.
