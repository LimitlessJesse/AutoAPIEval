Sample: 0 Invoker
----------------------------------------------------------------
1. `Object invoke(Object proxy, Method method, Object... args)`: This method invokes the specified method on the specified proxy with the specified arguments.
2. `void injectFault(Throwable t)`: This method injects a fault into the invocation chain.
3. `void setExecutor(Executor executor)`: This method sets the Executor to be used for invoking the endpoint.
4. `void setSource(Object source)`: This method sets the source of the invoker.
5. `void startRequest(MessageContext messageContext)`: This method starts the request with the given MessageContext.
================================================================

Sample: 1 Provider
----------------------------------------------------------------
1. `ServiceDelegate createServiceDelegate(URL wsdlDocumentLocation, QName serviceName, Class serviceClass)`: This method creates a new ServiceDelegate instance for the specified WSDL document location, service name, and service class.
2. `Endpoint createEndpoint(String bindingId, Object implementor)`: This method creates a new Endpoint instance with the specified binding ID and implementor object.
3. `ServiceDelegate createServiceDelegate(URL wsdlDocumentLocation, QName serviceName, Class serviceClass, WebServiceFeature... features)`: This method creates a new ServiceDelegate instance for the specified WSDL document location, service name, service class, and additional web service features.
4. `Endpoint createEndpoint(String bindingId, Object implementor, WebServiceFeature... features)`: This method creates a new Endpoint instance with the specified binding ID, implementor object, and additional web service features.
5. `ServiceDelegate createServiceDelegate(URL wsdlDocumentLocation, QName serviceName, Class serviceClass, WebServiceFeature[] features, Executor executor)`: This method creates a new ServiceDelegate instance for the specified WSDL document location, service name, service class, additional web service features, and executor for asynchronous operations.
================================================================

Sample: 2 ServiceDelegate
----------------------------------------------------------------
1. `Endpoint createEndpoint(String bindingId, Object implementor)`: This method creates a new endpoint for the specified binding ID and implementor object.
2. `Executor getExecutor()`: This method returns the executor used by this service delegate.
3. `ServiceDelegate newInstance()`: This method creates a new instance of the ServiceDelegate.
4. `void setExecutor(Executor executor)`: This method sets the executor to be used by this service delegate.
5. `void setHandlerResolver(HandlerResolver handlerResolver)`: This method sets the handler resolver for this service delegate.
================================================================
