Sample: 1 AudioFileWriter
----------------------------------------------------------------
1. `AudioFileFormat.Type[] getAudioFileTypes()`: This method returns an array of AudioFileFormat.Type objects indicating the audio file types supported for writing.
2. `int write(AudioInputStream stream, AudioFileFormat.Type fileType, File out)`: This method writes a stream of audio data provided by an AudioInputStream to the specified File in the specified file format.
3. `int write(AudioInputStream stream, AudioFileFormat.Type fileType, OutputStream out)`: This method writes a stream of audio data provided by an AudioInputStream to the specified OutputStream in the specified file format.
4. `boolean isFileTypeSupported(AudioFileFormat.Type fileType)`: This method returns true if the specified audio file type is supported for writing by this AudioFileWriter.
5. `boolean isFileTypeSupported(AudioFileFormat.Type fileType, AudioInputStream stream)`: This method returns true if the combination of the specified audio file type and AudioInputStream is supported for writing by this AudioFileWriter.
