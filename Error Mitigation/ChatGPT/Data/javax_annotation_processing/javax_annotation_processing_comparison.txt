Sample: 0 AbstractProcessor
----------------------------------------------------------------
1. `Set<String> getSupportedAnnotationTypes()`: This method returns a set of the fully qualified names of the annotation types supported by this processor.
2. `Set<String> getSupportedOptions()`: This method returns a set of the names of the supported options.
3. `SourceVersion getSupportedSourceVersion()`: This method returns the latest source version supported by this annotation processor.
4. `void init(ProcessingEnvironment processingEnv)`: This method initializes the processor with the processing environment.
5. `boolean process(Set<? extends TypeElement> annotations, RoundEnvironment roundEnv)`: This method processes a set of annotation types on type elements and returns true if the annotations were claimed by this processor.

----------------------------------------------------------------

2. `Completion of(TypeMirror type, String value)`: This method returns a completion of the specified type with the specified value.
3. `Completion of(TypeMirror type, String value, Element element)`: This method returns a completion of the specified type with the specified value and element.
4. `Completion of(TypeMirror type, String value, Element element, AnnotationMirror annotation)`: This method returns a completion of the specified type with the specified value, element, and annotation.
5. `Completion of(TypeMirror type, String value, Element element, AnnotationMirror annotation, AnnotationValue annotationValue)`: This method returns a completion of the specified type with the specified value, element, annotation, and annotation value.
