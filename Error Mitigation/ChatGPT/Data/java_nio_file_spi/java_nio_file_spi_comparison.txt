Sample: 0 FileSystemProvider
----------------------------------------------------------------
1. `FileSystemProvider provider()`: Returns the FileSystemProvider for this file system.
2. `FileSystem newFileSystem(Path path, Map<String, ?> env)`: Constructs a new FileSystem to access the contents of a file as a file system.
3. `FileSystem getFileSystem(URI uri)`: Returns the FileSystem for the given URI.
4. `Path getPath(URI uri)`: Converts the given URI to a Path object.
5. `void checkAccess(Path path, AccessMode... modes)`: Checks the access permissions to the file.

Sample: 1 FileTypeDetector
----------------------------------------------------------------
1. `String probeContentType(Path path)`: This method returns a string that represents the content type of the file specified by the given path.
2. `String toString()`: This method returns a string representation of the file type detector.
3. `boolean equals(Object obj)`: This method compares the specified object with this file type detector for equality.
4. `int hashCode()`: This method returns a hash code value for the file type detector.
5. `FileTypeDetector of()`: This method returns the default file type detector for the system.
