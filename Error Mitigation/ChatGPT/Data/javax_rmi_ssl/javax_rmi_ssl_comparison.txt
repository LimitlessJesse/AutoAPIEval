Sample: 0 SslRMIClientSocketFactory
----------------------------------------------------------------
1. `Socket createSocket(String host, int port)`: This method creates a socket connected to the specified host and port.
2. `String[] getEnabledCipherSuites()`: This method returns an array of the cipher suites enabled for use on this socket.
3. `String[] getEnabledProtocols()`: This method returns an array of the protocols enabled for use on this socket.
4. `boolean getNeedClientAuth()`: This method returns whether client authentication is required.
5. `boolean getWantClientAuth()`: This method returns whether client authentication is requested but not required.

Sample: 1 SslRMIServerSocketFactory
----------------------------------------------------------------
1. `ServerSocket createServerSocket(int port)`: This method creates a server socket on the specified port.
2. `ServerSocket createServerSocket(int port, int backlog)`: This method creates a server socket on the specified port with the specified backlog.
3. `ServerSocket createServerSocket(int port, int backlog, InetAddress ifAddress)`: This method creates a server socket on the specified port with the specified backlog and bind it to the specified local address.
4. `ServerSocket createServerSocket()`: This method creates a server socket without specifying any parameters.
5. `String[] getEnabledCipherSuites()`: This method returns an array of the cipher suites enabled for this server socket factory.
================================================================
