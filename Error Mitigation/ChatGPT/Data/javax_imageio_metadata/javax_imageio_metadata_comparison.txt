Sample: 0 IIOMetadata
----------------------------------------------------------------
1. `IIOMetadataFormat getMetadataFormat()`: This method returns the metadata format object that describes the structure of the metadata.
2. `Node getAsTree(String formatName)`: This method returns a node object that represents the metadata in the specified format.
3. `void mergeTree(String formatName, Node root)`: This method merges the given node into the current metadata tree for the specified format.
4. `void reset()`: This method resets the metadata to its initial state.
5. `boolean isReadOnly()`: This method returns true if the metadata is read-only, false otherwise.

4. `String[] getChildNames(String elementName)`: This method returns an array of the names of the children of the element with the specified name.
5. `String[] getAttributeNames(String elementName)`: This method returns an array of the names of the attributes of the element with the specified name.
