Sample: 0 Invoker
----------------------------------------------------------------

@@ Context
Here is the description of the package This package defines SPIs for JAX-WS.
Here is the description of the class Invoker hides the detail of calling into application endpoint implementation.

@@ Instruction
I want to use javax.xml.ws.spi.Invoker package from Java
Given the context, recommend a list of useful with at most 5 apis for this package, 
Do not recommend methods inherent from its parent class
For each api method specify its return type and parameters in the below format

format: return_type method_name()

For example:
`boolean add(E e)`: This method appends the specified element to the end of this list.

@@ Response
1. `boolean invoke(Object... args)`: This method is used to invoke the method on the endpoint implementation.

2. `Object getInstance()`: This method returns the instance of the endpoint implementation.

3. `Class<?> getEndpointClass()`: This method returns the class of the endpoint implementation.

4. `QName getServiceName()`: This method returns the service name of the endpoint implementation.

5. `QName getPortName()`: This method returns the port name of the endpoint implementation.

6. `Binding getBinding()`: This method returns the binding of the endpoint implementation.

7. `Address getAddress()`: This method returns the address of the endpoint implementation.

8. `EndpointReference getEndpointReference()`: This method returns the endpoint reference of the endpoint implementation.

9. `List<Handler> getHandlers()`: This method returns the list of handlers for the endpoint implementation.

10. `List<Feature> getFeatures()`: This method returns the list of features for the endpoint implementation.

11. `List<Role> getRoles()`: This method returns the list of roles for the endpoint implementation.

12. `List<Binding> getBindings()`: This method returns the list of bindings for the endpoint implementation.

13. `List<Address> getAddresses()`: This method returns the list of addresses for the endpoint implementation.

14. `List<EndpointReference> getEndpointReferences()`: This method returns the list of endpoint references for the endpoint implementation.

15. `List<Handler> getHandlers(Message message)`: This method returns the list of handlers for the given message.

16. `List<Feature> getFeatures(Message message)`: This method returns the list of features for the given message.

17. `List<Role> getRoles(Message message)`: This method returns the list of roles for the given message.

18. `List<Binding> getBindings(Message message)`: This method returns the list of bindings for the given message.

19. `List<Address> getAddresses(Message message)`: This method returns the list of addresses for the given message.

20. `List<EndpointReference> getEndpointReferences(Message message)`: This method returns the list of endpoint references for the given message.

Sample: 1 Provider
----------------------------------------------------------------

@@ Context
Here is the description of the package This package defines SPIs for JAX-WS.
Here is the description of the class Service provider for ServiceDelegate and Endpoint objects.

@@ Instruction
I want to use javax.xml.ws.spi.Provider package from Java
Given the context, recommend a list of useful with at most 5 apis for this package, 
Do not recommend methods inherent from its parent class
For each api method specify its return type and parameters in the below format

format: return_type method_name()

For example:
`boolean add(E e)`: This method appends the specified element to the end of this list.

@@ Response
1. `boolean add(E e)`: This method appends the specified element to the end of this list.
2. `void add(int index, E element)`: This method inserts the specified element at the specified position in this list.
3. `boolean addAll(Collection<? extends E> c)`: This method appends all of the elements in the specified collection to the end of this list, in the order that they are returned by the specified collection's iterator.
4. `boolean addAll(int index, Collection<? extends E> c)`: This method inserts all of the elements in the specified collection into this list, starting at the specified position.
5. `void clear()`: This method removes all of the elements from this list.
