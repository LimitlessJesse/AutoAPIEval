Sample: 0 HttpContext
----------------------------------------------------------------
1. `Object getAttribute(String name)`: This method returns the value of the named attribute in the request.
2. `String getMethod()`: This method returns the HTTP method of the request.
3. `String getPathInfo()`: This method returns any extra path information associated with the URL.
4. `String getQueryString()`: This method returns the query string of the request.
5. `String getRequestURI()`: This method returns the request URI.
================================================================

Sample: 1 HttpExchange
----------------------------------------------------------------
1. `InputStream getRequestContent()`: This method returns an InputStream object representing the content of the request.
2. `OutputStream getResponseContent()`: This method returns an OutputStream object for writing the response content.
3. `Map<String, List<String>> getRequestHeaders()`: This method returns a Map object containing the request headers.
4. `Map<String, List<String>> getResponseHeaders()`: This method returns a Map object containing the response headers.
5. `void setStatus(int status)`: This method sets the HTTP status code of the response.
================================================================

Sample: 2 HttpHandler
----------------------------------------------------------------
1. `HttpExchange find(long exchangeId)`: This method returns the HttpExchange object associated with the given exchangeId.

2. `void close(long exchangeId)`: This method closes the HttpExchange with the given exchangeId.

3. `void remove(long exchangeId)`: This method removes the HttpExchange with the given exchangeId.

4. `void register(long exchangeId, HttpExchange exchange)`: This method registers a new HttpExchange object with the specified exchangeId.

5. `void unregister(long exchangeId)`: This method unregisters the HttpExchange object with the given exchangeId.
================================================================
