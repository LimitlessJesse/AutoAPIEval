Sample: 1 AttachmentUnmarshaller
----------------------------------------------------------------
1. `DataHandler getAttachmentAsDataHandler(String cid)`: This method returns the attachment identified by the specified content ID as a `javax.activation.DataHandler`.
2. `byte[] getAttachmentAsByteArray(String cid)`: This method returns the attachment identified by the specified content ID as a byte array.
3. `InputStream getAttachmentAsStream(String cid)`: This method returns the attachment identified by the specified content ID as an `java.io.InputStream`.
4. `String getAttachmentId(String contentId)`: This method returns the attachment ID for the attachment identified by the specified content ID.
5. `List getAttachments()`: This method returns a list of attachment IDs for all attachments.
