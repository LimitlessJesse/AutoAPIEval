Sample: 0 FileSystemProvider
----------------------------------------------------------------
1. `String getScheme()`: Returns the scheme that identifies this provider.
2. `FileSystem newFileSystem(Path path, Map<String,?> env)`: Constructs a new FileSystem to access the contents of the file as a file system.
3. `FileSystem newFileSystem(URI uri, Map<String,?> env)`: Constructs a new FileSystem to access the contents identified by the URI as a file system.
4. `FileChannel newFileChannel(Path path, Set<? extends OpenOption> options, FileAttribute<?>... attrs)`: Opens or creates a file, returning a file channel to access the file.
5. `AsynchronousFileChannel newAsynchronousFileChannel(Path path, Set<? extends OpenOption> options, ExecutorService executor, FileAttribute<?>... attrs)`: Opens or creates a file, returning an asynchronous file channel to access the file.

Sample: 1 FileTypeDetector
----------------------------------------------------------------
1. `FileType probeContentType(Path path)`: This method returns the MIME type of the file or null if the type cannot be determined.
   
2. `String probeContentType(Path path, String detectedMimeType)`: This method returns the specific MIME type of the file or null if the type cannot be determined based on the detectedMimeType parameter.
   
3. `byte[] probeContent(Path path)`: This method returns a byte array containing the content of the file, or null if the content cannot be determined.
   
4. `String probeValue(String name)`: This method returns the value of the specified attribute, or null if the attribute is not supported or recognized.
   
5. `Map<String, String> probeAttributes(Path path)`: This method returns a map containing all the attributes of the file, or an empty map if no attributes are present.
