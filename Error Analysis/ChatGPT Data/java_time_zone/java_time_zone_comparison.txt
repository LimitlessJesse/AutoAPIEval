Sample: 0 ZoneOffsetTransition
----------------------------------------------------------------
1. `ZoneOffset getOffsetBefore(Instant instant)`: Returns the offset before the transition at the specified instant.
2. `ZoneOffset getOffsetAfter(Instant instant)`: Returns the offset after the transition at the specified instant.
3. `boolean isGap()`: Returns true if the transition is a gap, false if it is an overlap.
4. `ZoneOffsetTransition createTransition(Instant instant)`: Creates a transition instance for the specified instant.
5. `boolean isValidOffset(ZoneOffset offset)`: Returns true if the specified offset is valid for this transition.
================================================================

Sample: 1 ZoneOffsetTransitionRule
----------------------------------------------------------------
1. `ZoneOffsetTransition createTransition(int year)`: This method creates a ZoneOffsetTransition for the specified year.
2. `String getName()`: This method returns the name of the transition rule.
3. `Month getMonth()`: This method returns the month of the transition rule.
4. `DayOfWeek getDayOfWeek()`: This method returns the day of the week of the transition rule.
5. `TimeDefinition getTimeDefinition()`: This method returns the time definition of the transition rule.
================================================================

Sample: 2 ZoneRules
----------------------------------------------------------------
1. `ZoneOffsetTransition nextTransition(Instant instant)`: This method returns the next transition after the specified instant.
2. `List<ZoneOffsetTransition> getTransitions()`: This method returns a list of all the transitions in the zone rules.
3. `boolean isFixedOffset()`: This method returns true if the zone rules are fixed-offset, false otherwise.
4. `boolean isValidOffset(LocalDateTime localDateTime, ZoneOffset offset)`: This method checks if the offset is valid for the local date-time according to the zone rules.
5. `Period daylightSavingsAmount(Instant instant)`: This method returns the amount of daylight savings for the specified instant.
================================================================

Sample: 3 ZoneRulesProvider
----------------------------------------------------------------
1. `Set<String> getAvailableZoneIds()`: This method returns a set of all available zone IDs.
2. `ZoneRules getRules(String zoneId)`: This method returns the rules for the zone ID.
3. `ZoneOffset getOffset(Instant instant)`: This method returns the zone offset for the given instant.
4. `ZoneOffsetTransition getTransition(Instant instant)`: This method returns the next transition after the specified instant.
5. `NavigableMap<Instant, ZoneOffset> getTransitions(String zoneId)`: This method returns a navigable map of all transitions for the specified zone ID.
