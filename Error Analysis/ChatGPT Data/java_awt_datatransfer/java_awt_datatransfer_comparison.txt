Sample: 0 Clipboard
----------------------------------------------------------------
1. `void setContents(Transferable contents, ClipboardOwner owner)`: This method sets the current contents of the clipboard to the specified transferable object with the specified owner.
2. `Transferable getContents(Object requestor)`: This method gets the current contents of the clipboard in the form of a Transferable object with the specified requestor.
3. `String getName()`: This method gets the name of the clipboard owner.
4. `void setName(String name)`: This method sets the name of the clipboard.
5. `void addFlavorListener(FlavorListener listener)`: This method adds a FlavorListener to listen for changes to the available DataFlavor objects on this clipboard.
================================================================

Sample: 1 DataFlavor
----------------------------------------------------------------
1. `String getMimeType()`: This method returns the MIME type string for this DataFlavor.
2. `Class<?> getRepresentationClass()`: This method returns the class used to represent the data for this DataFlavor.
3. `String getHumanPresentableName()`: This method returns a human presentable name for the data, localized to the default locale.
4. `int hashCode()`: This method returns a hash code value for the object.
5. `boolean equals(Object obj)`: This method compares this DataFlavor to the specified object.
================================================================

Sample: 2 FlavorEvent
----------------------------------------------------------------
1. `FlavorEvent getFlavorEvent()`: This method returns the FlavorEvent associated with this event.
2. `DataFlavor[] getTransferDataFlavors()`: This method returns an array of DataFlavor objects representing the canonical format(s) of the data transferred in this event.
3. `boolean isFlavorChanged()`: This method returns whether the set of DataFlavors changed in this event.
4. `boolean isLocalTransfer()`: This method returns whether the transfer is local to the same Java Virtual Machine.
5. `boolean isMimeTypeEqual(String mimeType)`: This method returns whether the specified MIME Type is equal to the MIME Type of the DataFlavor of this current event.
================================================================

Sample: 3 StringSelection
----------------------------------------------------------------
1. `String getTransferDataFlavors()`: This method returns an array of DataFlavor objects representing the flavors the data can be provided in.
2. `void setString(String data)`: This method sets the String to be transferred to the specified String.
3. `void addFlavorListener(FlavorListener listener)`: This method adds a FlavorListener to this object to listen for FlavorEvents.
4. `void removeFlavorListener(FlavorListener listener)`: This method removes a FlavorListener from this object.
5. `boolean isDataFlavorSupported(DataFlavor flavor)`: This method returns whether or not the specified DataFlavor is supported for this object.
================================================================

Sample: 4 SystemFlavorMap
----------------------------------------------------------------
1. `void addFlavorForUnencodedNative(String nat, DataFlavor flav)`: This method adds a mapping from a native platform-specific format name to a DataFlavor.
2. `void addFlavorForNatives(DataFlavor flav, String[] nat): This method adds mapping from a DataFlavor to several native platform-specific formats.
3. `void resetFlavors()`: This method resets the SystemFlavorMap to its initial state.
4. `void removeFlavorForUnencodedNative(String nat, DataFlavor flav)`: This method removes the mapping from a native platform-specific format name to a DataFlavor.
5. `void removeFlavorForNatives(DataFlavor flav, String[] nat)`: This method removes the mapping from a DataFlavor to several native platform-specific formats.
================================================================
