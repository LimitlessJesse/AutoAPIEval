Sample: 1 DefaultHandler2_1
----------------------------------------------------------------
void startCDATA() method is used to indicate the start of a CDATA section in a SAX event handler. Here is a code snippet demonstrating the use of this method:

```java
import org.xml.sax.ext.DefaultHandler2;

public class Main extends DefaultHandler2 {

    @Override
    public void startCDATA() {
        System.out.println("Start of CDATA section");
    }

    public static void main(String[] args) {
        Main handler = new Main();
        handler.startCDATA();
    }
}
```

Sample: 3 DefaultHandler2_3
----------------------------------------------------------------
`void startDTD(String name, String publicId, String systemId)`: This method is called at the start of the DTD.

Code snippet:
```java
import org.xml.sax.ext.DefaultHandler2;

public class Main extends DefaultHandler2 {
    @Override
    public void startDTD(String name, String publicId, String systemId) {
        System.out.println("Start of DTD:");
        System.out.println("Name: " + name);
        System.out.println("Public ID: " + publicId);
        System.out.println("System ID: " + systemId);
    }

    public static void main(String[] args) {
        Main handler = new Main();
        handler.startDTD("myDTD", "-//W3C//DTD HTML 4.01 Transitional//EN", "http://www.w3.org/TR/html4/loose.dtd");
    }
}
```
