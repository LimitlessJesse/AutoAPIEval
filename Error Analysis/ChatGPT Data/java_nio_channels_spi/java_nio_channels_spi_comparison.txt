Sample: 3 AbstractSelector
----------------------------------------------------------------
1. `SelectionKey register(SelectableChannel ch, int ops, Object att)`: Register the given channel with this selector, returning a selection key.
2. `int select()`: Performs a blocking selection operation.
3. `int selectNow()`: Performs a non-blocking selection operation.
4. `int select(long timeout)`: Performs a selection operation with a specified timeout.
5. `Set<SelectionKey> keys()`: Returns the selected-key set of this selector.
