Sample: 0 AbstractUndoableEdit
----------------------------------------------------------------
1. `boolean canRedo()`: This method returns whether there are any edits to redo.
2. `boolean canUndo()`: This method returns whether there are any edits to undo.
3. `String getPresentationName()`: This method returns a String that can be displayed to describe the undo or redo operation.
4. `String getRedoPresentationName()`: This method returns a String that describes the redo operation.
5. `String getUndoPresentationName()`: This method returns a String that describes the undo operation.
================================================================

Sample: 1 CompoundEdit
----------------------------------------------------------------
1. `void end()`: This method ends the edit sequence and removes all edits that have been undone or redone.
   
2. `void die()`: This method invokes die() on each subedit, sending the last one first.

3. `boolean isInProgress()`: This method returns true if this edit is in progress.

4. `void undo()`: This method undoes all of the edits.

5. `void redo()`: This method redoes all of the edits.
================================================================

Sample: 2 StateEdit
----------------------------------------------------------------
1. `boolean canRedo()`: This method returns a boolean value indicating whether it is possible to redo the last undone change.
2. `void redo()`: This method redoes the last undone change.
3. `boolean canUndo()`: This method returns a boolean value indicating whether it is possible to undo the last change.
4. `void undo()`: This method undoes the last change.
5. `String getPresentationName()`: This method returns the presentation name of the edit.
================================================================

Sample: 3 UndoableEditSupport
----------------------------------------------------------------
1. `void addUndoableEditListener(UndoableEditListener listener)`: Adds a listener to this UndoableEditSupport.
2. `void removeUndoableEditListener(UndoableEditListener listener)`: Removes a listener from this UndoableEditSupport.
3. `void postEdit(UndoableEdit edit)`: Notifies all listeners that an undoable edit has occurred.
4. `void end()`: Indicates the end of a series of edits.
5. `boolean isSignificant()`: Returns true if this edit is considered significant.
================================================================

Sample: 4 UndoManager
----------------------------------------------------------------
1. `void undo()`: This method undoes the last edit operation in the undo manager.
2. `void redo()`: This method redoes the last undone edit operation in the undo manager.
3. `boolean canUndo()`: This method returns a boolean indicating whether an undo operation is possible.
4. `boolean canRedo()`: This method returns a boolean indicating whether a redo operation is possible.
5. `void discardAllEdits()`: This method discards all edits in the undo manager.
================================================================
