Sample: 0 NamingEvent
----------------------------------------------------------------
1. `Object getNewBinding()`: This method returns the new binding that triggered the event.
   
2. `Object getOldBinding()`: This method returns the old binding that was replaced by the new binding.

3. `Object getSource()`: This method returns the source of the event.

4. `int getState()`: This method returns the state of the event.

5. `String toString()`: This method returns a string representation of the NamingEvent.

Sample: 1 NamingExceptionEvent
----------------------------------------------------------------
1. `Object getSource()`: This method returns the object on which the Event initially occurred.
2. `Throwable getException()`: This method returns the exception that was generated or caused the event.
3. `Hashtable getEnvironment()`: This method returns the environment of the context naming exception being generated.
4. `String toString()`: This method returns a string representation of the NamingExceptionEvent.
5. `void dispatch(NamingExceptionListener listener)`: This method dispatches the event to the specified listener.
