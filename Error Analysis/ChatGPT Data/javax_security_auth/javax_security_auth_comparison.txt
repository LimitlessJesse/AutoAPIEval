Sample: 0 AuthPermission
----------------------------------------------------------------
1. `String getActions()`: This method returns the actions granted in the permission.
2. `String getActions(String target)`: This method returns the actions granted to the specified target in the permission.
3. `void setActions(String actions)`: This method sets the actions in the permission.
4. `void setActions(String target, String actions)`: This method sets the actions granted to the specified target in the permission.
5. `boolean implies(Permission permission)`: This method checks if the specified permission is implied by this permission.
================================================================

Sample: 1 Policy
----------------------------------------------------------------
1. `PermissionCollection getPermissions(ProtectionDomain domain)`: This method returns a PermissionCollection object containing the set of permissions granted to the specified ProtectionDomain.

2. `void refresh()`: This method refreshes the policy configuration by discarding any cached policy data.

3. `void setPolicy(Policy policy)`: This method sets the Policy object that will be used by the Policy class.

4. `PermissionCollection getPermissions(CodeSource codesource)`: This method returns a PermissionCollection object containing the set of permissions granted to code from the specified CodeSource.

5. `boolean implies(ProtectionDomain domain, Permission permission)`: This method checks if the specified permissions are implied by the policy for the given ProtectionDomain.
================================================================

Sample: 2 PrivateCredentialPermission
----------------------------------------------------------------
1. `String [] getActions()`: This method returns the actions associated with this PrivateCredentialPermission.
2. `java.util.Set<java.security.Principal> getPrincipals()`: This method returns the set of Principals associated with this PrivateCredentialPermission.
3. `java.lang.String getCredentialClass()`: This method returns the credential class name of this PrivateCredentialPermission.
4. `java.lang.String toString()`: This method returns a string representation of this PrivateCredentialPermission.
5. `boolean implies(java.security.Permission permission)`: This method checks if the specified permission is implied by this PrivateCredentialPermission.
================================================================

Sample: 4 SubjectDomainCombiner
----------------------------------------------------------------
1. `void setSubject(Subject subject)`: This method sets the Subject for this SubjectDomainCombiner.
2. `Subject getSubject()`: This method returns the current Subject associated with this SubjectDomainCombiner.
3. `ProtectionDomain[] combine(ProtectionDomain[] currentDomains, ProtectionDomain[] assignedDomains)`: This method combines the ProtectionDomains of the currentDomains.
4. `Object clone()`: This method creates and returns a copy of this SubjectDomainCombiner instance.
5. `boolean equals(Object obj)`: This method compares the specified object with this SubjectDomainCombiner for equality.
