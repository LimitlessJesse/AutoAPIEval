Sample: 0 DefaultLoaderRepository
----------------------------------------------------------------
1. `ClassLoader getClassLoader(String name)`: This method returns the class loader for the specified name.
2. `ClassLoader getClassLoader(String name, ObjectName loaderName)`: This method returns the class loader for the specified name associated with the given loader name.
3. `List<ClassLoader> getClassLoaders()`: This method returns a list of all registered class loaders.
4. `void addLoader(Object loader, String name, Object context)`: This method registers a new class loader with the specified name and context.
5. `void removeLoader(ObjectName name)`: This method unregisters the class loader with the specified name.
================================================================

Sample: 1 MLet
----------------------------------------------------------------
1. `void addURL(URL url)`: This method adds the specified URL to the list of URLs from which this MLet will load classes.
2. `int hashCode()`: This method returns the hash code value for this MLet.
3. `void start()` : This method starts the MLet. 
4. `void stop()`: This method stops the MLet.
5. `void load()`: This method loads all the MBeans specified in the MLet URL.
================================================================

Sample: 2 MLetContent
----------------------------------------------------------------
1. `void addMBeans()`: This method adds the MBeans specified in the MLet document to the MBean server.
2. `String getMBeansFromURL(String url)` : This method retrieves the MBeans specified in the given URL and returns them as a String.
3. `List<URL> getURLs()`: This method returns a list of the URLs from which the MBeans were loaded.
4. `void removeAllMBeans()`: This method removes all MBeans loaded by this MLet instance from the MBean server.
5. `void removeNotificationListener(NotificationListener listener, NotificationFilter filter, Object handback)`: This method removes a listener from the list of listeners for notifications emitted by MBeans loaded by this MLet instance.
================================================================

Sample: 3 PrivateMLet
----------------------------------------------------------------
1. `void addMBeans(MBeanServer mbs)`: This method adds the MBeans specified in the PrivateMLet to the given MBean server.
2. `MBeanServer getMBeanServer()`: This method returns the MBean server where the MBeans specified in the PrivateMLet are registered.
3. `void removeMBeans(MBeanServer mbs)`: This method removes the MBeans specified in the PrivateMLet from the given MBean server.
4. `Set<ObjectName> queryNames(ObjectName name, QueryExp query)`: This method queries the MBean server for MBeans matching the specified object name pattern and query expression.
5. `MBeanInfo getMBeanInfo(ObjectName name)`: This method retrieves the meta information for the MBean specified by the object name.
================================================================
