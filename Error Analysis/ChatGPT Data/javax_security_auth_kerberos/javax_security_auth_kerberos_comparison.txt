Sample: 0 DelegationPermission
----------------------------------------------------------------
1. `void checkPermission(java.security.Permission perm)`: This method checks if the Permission object passed as a parameter implies the permissions specified in this DelegationPermission.
2. `String getActions()`: This method returns the actions permitted for this DelegationPermission.
3. `String getName()`: This method returns the name of this DelegationPermission.
4. `boolean implies(java.security.Permission p)`: This method checks if the specified permission is "implied" by this DelegationPermission.
5. `boolean implies(PermissionCollection permissions)`: This method checks if the specified PermissionCollection is "implied" by this DelegationPermission.
