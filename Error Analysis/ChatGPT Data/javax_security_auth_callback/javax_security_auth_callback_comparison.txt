Sample: 0 ChoiceCallback
----------------------------------------------------------------
1. `int[] getSelectedIndexes()`: Returns an array of integer values representing the indexes of the selected choices.
2. `ChoiceCallback.Option[] getOptions()`: Returns an array of ChoiceCallback.Option objects representing the available choices.
3. `void setSelectedIndexes(int[] selectedIndexes)`: Sets the indexes of the selected choices based on the input array of integers.
4. `void setSelections(ChoiceCallback.Option[] selectedOptions)`: Sets the selected choices based on the input array of ChoiceCallback.Option objects.
5. `String getPrompt()`: Returns the prompt message associated with the ChoiceCallback.
================================================================

----------------------------------------------------------------
1. `ConfirmationCallback(boolean allowCancel)`: Constructor that creates a ConfirmationCallback with a boolean parameter for allowing cancellation.
2. `void setSelectedIndex(int index)`: Sets the selected choice index for the ConfirmationCallback.
3. `int getSelectedIndex()`: Returns the selected choice index for the ConfirmationCallback.
4. `void setDefaultOption(int defaultOption)`: Sets the default option index for the ConfirmationCallback.
5. `int getDefaultOption()`: Returns the default option index for the ConfirmationCallback.

Sample: 4 PasswordCallback
----------------------------------------------------------------
1. `char[] getPassword()`: This method returns the password as a character array.
2. `void setPassword(char[] password)`: This method sets the password to the specified character array.
3. `boolean isEchoOn()`: This method returns a boolean indicating whether password echo is enabled.
4. `void setEchoOn(boolean echoOn)`: This method sets whether password echo should be enabled or disabled.
5. `String getPrompt()`: This method returns the prompt used to request the password.
================================================================

Sample: 5 TextInputCallback
----------------------------------------------------------------

4. `void setPrompt(String prompt)`: This method sets the prompt to be displayed to the user.
5. `void clearText()`: This method clears the text entered by the user.
