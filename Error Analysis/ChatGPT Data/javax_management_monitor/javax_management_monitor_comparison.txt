Sample: 0 CounterMonitor
----------------------------------------------------------------
1. `void setThreshold(Number value)`: Sets the threshold value for the counter.
2. `Number getThreshold()`: Returns the current threshold value for the counter.
3. `void setOffset(Number value)`: Sets the offset value for the counter.
4. `Number getOffset()`: Returns the current offset value for the counter.
5. `void setDifferenceMode(boolean value)`: Sets whether the counter is in difference mode or not.
================================================================

Sample: 1 GaugeMonitor
----------------------------------------------------------------
1. `void start()`: This method starts the monitor.
2. `void stop()`: This method stops the monitor.
3. `Object getDerivedGauge()`: This method returns the derived gauge value.
4. `Long getGranularityPeriod()`: This method returns the granularity period.
5. `void setThresholds(Number highValue, Number lowValue)`: This method sets the high and low threshold values for the gauge monitor.

Sample: 2 Monitor
----------------------------------------------------------------
1. `void start()`: This method starts the monitoring.
2. `void stop()`: This method stops the monitoring.
3. `Number getDerivedGauge()`: This method returns the derived gauge value.
4. `String getObservedAttribute()`: This method returns the observed attribute.

Sample: 3 MonitorNotification
----------------------------------------------------------------
1. `String getObservedAttribute()`: This method returns the observed attribute of the notification.
2. `String getTrigger()`: This method returns the trigger that caused the notification.
3. `Object getDerivedGauge()`: This method returns the derived gauge value contained in the notification.
4. `String getDerivedGaugeType()`: This method returns the type of the derived gauge value.
5. `MonitorNotification clone()`: This method creates and returns a copy of this object.
================================================================

Sample: 4 StringMonitor
----------------------------------------------------------------
1. `String getDerivedGauge()`: Returns a string representing the derived gauge value.
2. `String getDerivedGauge(ObjectName object)`: Returns a string representing the derived gauge value for a specific object name.
3. `String getDerivedGaugeTimeStamp()`: Returns a string representing the timestamp when the derived gauge value was calculated.
4. `String getMonitor()`: Returns a string representing the monitor MBean name.
5. `void setStringToCompare(String value)`: Sets the string value to compare against for monitoring purposes.
