Sample: 0 LocateRegistry
----------------------------------------------------------------
1. `Registry createRegistry(int port)`: Creates and exports a registry on the specified port.
2. `Registry getRegistry()`: Returns a reference to the registry on the localhost at the default registry port.
3. `Registry createRegistry()`: Creates and exports a registry on the default registry port.
4. `void bind(String name, Remote obj)`: Binds a remote reference to the specified name in this registry.
5. `Remote lookup(String name)`: Returns the remote reference bound to the specified name in this registry.
