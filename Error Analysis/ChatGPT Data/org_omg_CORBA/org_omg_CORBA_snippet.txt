Sample: 6 ORB_3

`void run()`: This method starts the ORB processing events and handling requests. It blocks until the ORB is shut down.

Code snippet:
```java
import org.omg.CORBA.ORB;

public class Main {

        ORB orb = ORB.init(args, null);
        orb.run();
    }
}
```

Sample: 7 ORB_4
----------------------------------------------------------------
`void shutdown(boolean waitForCompletion)`: This method initiates the shutdown process of the ORB. If `waitForCompletion` is `true`, then the method blocks until the ORB completes the shutdown process.

Code snippet:
```java
import org.omg.CORBA.ORB;

public class Main {
    public static void main(String[] args) {
        ORB orb = ORB.init(args, null);
        
        // Perform some ORB operations
        
        orb.shutdown(true); // Initiates shutdown and waits for completion
        
        System.out.println("ORB shutdown completed.");
    }
}
```
