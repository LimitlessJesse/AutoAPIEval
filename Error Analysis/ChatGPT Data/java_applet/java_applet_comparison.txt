Sample: 0 Applet
----------------------------------------------------------------
1. `AudioClip getAudioClip(URL url)`: This method returns an AudioClip object from the specified URL.
2. `void play(URL url)`: This method plays the audio clip from the specified URL.
3. `void showStatus(String msg)`: This method displays a status string in the applet.
4. `void init()`: This method is called by the browser or applet viewer to inform this applet that it has been loaded into the system.
5. `void start()`: This method is called by the browser or applet viewer to inform this applet that it should start its execution.
