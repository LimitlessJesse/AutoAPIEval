Sample: 0 MarshalledObject
----------------------------------------------------------------
1. `Object get()`: Returns the serialized array of objects contained in this MarshalledObject.
2. `boolean equals(Object obj)`: Indicates whether some other object is "equal to" this one.
3. `int hashCode()`: Returns a hash code value for the object.
4. `String toString()`: Returns a string representation of the object.
5. `MarshalledObject<T> clone()`: Returns a clone of this MarshalledObject instance.
================================================================

Sample: 1 Naming
----------------------------------------------------------------
1. `T lookup(String name)`: This method looks up and returns the remote object bound to the specified name.
2. `void rebind(String name, Remote obj)`: This method binds a remote object to the specified name.
3. `String[] list(String name)`: This method lists the names of the remote objects bound to the specified name.
4. `void unbind(String name)`: This method removes the binding for the specified name.
5. `void bind(String name, Remote obj)`: This method binds a remote object to the specified name.
================================================================

Sample: 2 RMISecurityManager
----------------------------------------------------------------
1. `void checkAccess(RemoteObject obj)`: This method checks whether the caller has permission to access the remote object.
2. `void checkConnect(String host, int port)`: This method checks whether the caller has permission to connect to the specified host and port.
3. `void checkPermission(Permission perm)`: This method checks whether the caller has been granted the specified permission.
4. `void checkPropertiesAccess()`: This method checks whether the caller has permission to access system properties.
5. `void checkAccept(String host, int port)`: This method checks whether the caller has permission to accept connections on the specified host and port.
================================================================
