Sample: 0 Book
----------------------------------------------------------------
1. `void append(Printable printable)`: This method appends a Printable object to the end of the book.
2. `void insert(Printable printable, int pageIndex)`: This method inserts a Printable object at the specified page index in the book.
3. `Printable getPage(int pageIndex)`: This method returns the Printable object at the specified page index in the book.
4. `int getNumberOfPages()`: This method returns the total number of pages in the book.
5. `void remove(Printable printable)`: This method removes the specified Printable object from the book.
================================================================

Sample: 1 PageFormat
----------------------------------------------------------------
1. `void setOrientation(int orientation)`: Set the orientation of the page (either PORTRAIT or LANDSCAPE).
   
2. `int getOrientation()`: Return the orientation of the page.
   
3. `void setPaper(Paper paper)`: Set the size and imageable area of the paper for this PageFormat.
   
4. `Paper getPaper()`: Get the Paper object that represents the size and imageable area of the page.
   
5. `void setMargin(double top, double left, double bottom, double right)`: Set the margins of the page.
================================================================

Sample: 2 Paper
----------------------------------------------------------------
1. `double getHeight()`: Returns the height of the paper.
2. `double getWidth()`: Returns the width of the paper.
3. `void setSize(double width, double height)`: Sets the width and height of the paper.
4. `void setImageableArea(double x, double y, double width, double height)`: Sets the imageable area of the paper.
5. `void setImageableArea(Rectangle2D imageableArea)`: Sets the imageable area of the paper using a Rectangle2D object.
================================================================

Sample: 3 PrinterJob
----------------------------------------------------------------
1. `void setPrintable(Printable painter)`: This method sets the Printable object to use for printing.
2. `PageFormat getPageFormat()`: This method returns the PageFormat object that describes the printable area of the page.
3. `PrintService getPrintService()`: This method returns the PrintService object that represents the print service that the job will use.
4. `void setCopies(int copies)`: This method sets the number of copies to be printed.
5. `void print()`: This method initiates a print job.
================================================================
