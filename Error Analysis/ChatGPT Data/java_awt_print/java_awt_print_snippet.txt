Sample: 1 PageFormat_1
----------------------------------------------------------------
`void setOrientation(int orientation)`: This method sets the orientation of the page to either PORTRAIT or LANDSCAPE.

Code snippet:
```java
import java.awt.print.PageFormat;

public class Main {
    public static void main(String[] args) {
        PageFormat pageFormat = new PageFormat();
        int orientation = PageFormat.PORTRAIT; // or PageFormat.LANDSCAPE
        pageFormat.setOrientation(orientation);
        System.out.println("Page orientation set to: " + orientation);
    }
}
```

Sample: 3 PageFormat_3
----------------------------------------------------------------
`void setPaper(Paper paper)`: This method sets the Paper object, which describes the size and orientation of the page.

Code snippet:
```java
import java.awt.print.PageFormat;
import java.awt.print.Paper;

public class Main {
    public static void main(String[] args) {
        PageFormat pageFormat = new PageFormat();
        Paper paper = new Paper();
        paper.setSize(595.22, 841.85); // A4 size in points
        paper.setImageableArea(0, 0, paper.getWidth(), paper.getHeight()); // Imageable area
        pageFormat.setPaper(paper);
        
        System.out.println("Page Width: " + paper.getWidth());
        System.out.println("Page Height: " + paper.getHeight());
    }
}
```

Sample: 12 PrinterJob_5
----------------------------------------------------------------
void print(): This method initiates a print job with the current job attributes. 

Code snippet:
```java
import java.awt.print.PrinterJob;

public class Main {
    public static void main(String[] args) {
        PrinterJob job = PrinterJob.getPrinterJob();
        if (job.printDialog()) {
            try {
                job.print();
            } catch (Exception e) {
                System.out.println("Error printing: " + e.getMessage());
            }
        }
    }
}
```
