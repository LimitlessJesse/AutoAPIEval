Sample: 0 ServerSocketFactory
----------------------------------------------------------------
1. `ServerSocketFactory getDefault()`: Returns the ServerSocketFactory that acts as the default factory for all server sockets.
2. `ServerSocket createServerSocket(int port)`: Creates a server socket bound to the specified port.
3. `ServerSocket createServerSocket(int port, int backlog)`: Creates a server socket bound to the specified port with a specified backlog.
4. `ServerSocket createServerSocket(int port, int backlog, InetAddress ifAddress)`: Creates a server socket bound to the specified port with a specified backlog and bound to a specific local address.
5. `Socket createSocket()`: Creates a socket.
================================================================

Sample: 1 SocketFactory
----------------------------------------------------------------
1. `Socket createSocket()`: This method creates a socket without connecting it to a remote host.
2. `Socket createSocket(String host, int port)`: This method creates a socket and connects it to the specified remote host on the specified port.
3. `Socket createSocket(InetAddress address, int port)`: This method creates a socket and connects it to the specified remote InetAddress on the specified port.
4. `Socket createSocket(String host, int port, InetAddress localHost, int localPort)`: This method creates a socket and connects it to the specified remote host on the specified port using the specified local network address and port.
5. `Socket createSocket(InetAddress address, int port, InetAddress localHost, int localPort)`: This method creates a socket and connects it to the specified remote InetAddress on the specified port using the specified local network address and port.
================================================================
