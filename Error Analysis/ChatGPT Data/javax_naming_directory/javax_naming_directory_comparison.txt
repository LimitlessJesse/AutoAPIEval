Sample: 0 BasicAttribute
----------------------------------------------------------------
1. `void add(String attrID)`: This method adds a new attribute with the specified attribute ID to this BasicAttribute.
2. `void add(String attrID, Object val)`: This method adds a new attribute with the specified attribute ID and value to this BasicAttribute.
3. `void remove(String attrID)`: This method removes the attribute with the specified attribute ID from this BasicAttribute.
4. `void clear()`: This method removes all attributes from this BasicAttribute.
5. `int size()`: This method returns the number of attributes in this BasicAttribute.
================================================================

Sample: 1 BasicAttributes
----------------------------------------------------------------
1. `void put(String attrID, Object val)`: This method sets the attribute specified by the attrID with the given value.
2. `Attribute get(String attrID)`: This method returns the attribute associated with the specified ID.
3. `void remove(String attrID)`: This method remove the attribute specified by the attrID from the attribute set.
4. `boolean contains(String attrID)`: This method returns true if the attribute set contains an attribute with the specified ID.
5. `NamingEnumeration getAll()`: This method returns an enumeration of all the attributes in the attribute set.

Sample: 2 InitialDirContext
----------------------------------------------------------------
1. `Attributes getAttributes(String name)`: This method retrieves the attributes associated with a named object.
2. `NamingEnumeration<NameClassPair> list(Name name)`: This method lists the names bound in the named context.
3. `void rebind(Name name, Object obj)`: This method rebinds the specified object to the specified name in this context.
4. `void destroySubcontext(String name)`: This method removes the named object and its associated binding from the namespace.
5. `void unbind(String name)`: This method removes an object from the namespace.

Sample: 3 ModificationItem
----------------------------------------------------------------
1. `int getModificationOp()`: This method returns the modification operation code for the modification.
2. `Name getName()`: This method returns the name of the attribute to be modified.
3. `Attributes getAttributes()`: This method returns the attributes associated with the modification.
4. `ModificationItem(ModificationItem[] items)`: Constructor that creates a new ModificationItem with the specified array of ModificationItems.
5. `ModificationItem(int mod_op, Attribute attr)`: Constructor that creates a new ModificationItem with the specified modification operation and attribute.

Sample: 4 SearchControls
----------------------------------------------------------------
1. `int getTimeLimit()`: Returns the maximum number of milliseconds that the search operation is allowed to take.
2. `void setTimeLimit(int timeLimit)`: Sets the maximum number of milliseconds that the search operation is allowed to take.
3. `int getScope()`: Returns the scope of the search.
4. `void setScope(int scope)`: Sets the scope of the search.
5. `int getCountLimit()`: Returns the maximum number of entries that the search operation is allowed to return.
================================================================

Sample: 5 SearchResult
----------------------------------------------------------------
1. `Attributes getAttributes()`: This method returns the attributes of the object that was found during the search.

2. `String getName()`: This method returns the name of the object that was found during the search.

3. `String getDN()`: This method returns the distinguished name of the object that was found during the search.

4. `boolean isRelative()`: This method returns true if the search result is relative, false otherwise.

5. `DirContext getCtx()`: This method returns the directory context of the object that was found during the search.
