Sample: 1 Provider
----------------------------------------------------------------
1. `javax.xml.ws.spi.Provider.ServiceDelegate createServiceDelegate(URL wsdlDocumentLocation, QName serviceName, Class<javax.xml.ws.Service> serviceClass)`: This method creates a new ServiceDelegate instance for the given WSDL document location, service name, and service class.
   
2. `javax.xml.ws.spi.Provider.createPortDelegate(EndpointReference epr, Class<T> clazz)`: This method creates a new PortDelegate instance for the given EndpointReference and class.

3. `ServiceDelegate createServiceDelegate(URL wsdlDocumentLocation, QName serviceName, Class<javax.xml.ws.Service> serviceClass, WebServiceFeature... features)`: This method creates a new ServiceDelegate instance for the given WSDL document location, service name, service class, and additional web service features.

4. `PortInfo createPortInfo(WSDLPort port)`: This method creates a new PortInfo instance for the given WSDLPort.

5. `void disptach(Object arg0, MethodHandler arg1, Map<String, Object> arg2)`: This method dispatches the method call on the handler.
