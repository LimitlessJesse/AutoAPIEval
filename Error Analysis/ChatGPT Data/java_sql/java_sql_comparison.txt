----------------------------------------------------------------

4. `void setLoginTimeout(int seconds)`: This method sets the maximum time in seconds that a driver will wait while attempting to log in to a database.
5. `Driver getDriver(String url)`: This method attempts to locate a driver that understands the given URL.
