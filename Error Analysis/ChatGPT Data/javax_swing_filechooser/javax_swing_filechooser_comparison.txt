Sample: 0 FileFilter
----------------------------------------------------------------
1. `boolean accept(File f)`: Returns a boolean indicating whether the given file is accepted by this filter.
2. `String getDescription()`: Returns the description of the file filter.
3. `void setDescription(String description)`: Sets the description of the file filter.
4. `String getDescription(File f)`: Returns a string that is the description of the file.
5. `boolean equals(Object obj)`: Indicates whether some other object is "equal to" this file filter.
================================================================

Sample: 1 FileNameExtensionFilter
----------------------------------------------------------------
1. `String getDescription()`: This method returns the description of the file filter.
2. `String getExtension()`: This method returns the file extension used by the file filter.
3. `boolean accept(File f)`: This method determines if the given file is accepted by the file filter.
4. `void setDescription(String desc)`: This method sets the description of the file filter.
5. `void setExtension(String ext)`: This method sets the file extension to be used by the file filter.
================================================================

Sample: 2 FileSystemView
----------------------------------------------------------------
1. `File[] getRoots()`: This method returns an array of File objects representing the file system roots.
2. `File getHomeDirectory()`: This method returns a File object representing the user's home directory.
3. `Boolean isRoot(File f)`: This method checks if the given File object is a root in the file system.
4. `File getDefaultDirectory()`: This method returns a File object representing the default directory that the file chooser should show upon opening.
5. `File createNewFolder(File containingDir)`: This method creates a new folder in the specified directory and returns a File object representing the newly created folder.
================================================================

Sample: 3 FileView
----------------------------------------------------------------
1. `String getName(File f)`: This method returns the name of the file.
2. `String getTypeDescription(File f)`: This method returns a description for the type of the file.
3. `String getDescription(File f)`: This method returns the description of the file.
4. `Icon getIcon(File f)`: This method returns the icon for the file.
5. `Boolean isTraversable(File f)`: This method returns whether the file is traversable or not.
================================================================
