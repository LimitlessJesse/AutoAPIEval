Sample: 0 DocFlavor_1
----------------------------------------------------------------
`String getMediaType()`: This method retrieves the MIME type string corresponding to this doc flavor.

Code snippet:
```java
import javax.print.DocFlavor;

public class Main {
    public static void main(String[] args) {
        DocFlavor docFlavor = DocFlavor.BYTE_ARRAY.AUTOSENSE;
        String mediaType = docFlavor.getMediaType();
        System.out.println("Media Type: " + mediaType);
    }
}
```

Sample: 4 StreamPrintService_2
----------------------------------------------------------------
`OutputStream getOutputStream()`: This method retrieves an output stream for communicating with the printer. 

Code snippet:
```java
import javax.print.StreamPrintService;

public class Main {
    public static void main(String[] args) {
        StreamPrintService streamPrintService = new StreamPrintService();
        OutputStream outputStream = streamPrintService.getOutputStream();
        // You can now write to outputStream to communicate with the printer
    }
}
```
