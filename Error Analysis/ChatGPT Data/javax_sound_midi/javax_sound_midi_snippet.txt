Sample: 3 MetaMessage_3
----------------------------------------------------------------
Code snippet:
```java
import javax.sound.midi.MetaMessage;

public class Main {
    public static void main(String[] args) {
        MetaMessage message = new MetaMessage();
        int type = 1; // Example type value
        byte[] data = {0x01, 0x02, 0x03}; // Example byte array data
        int length = data.length; // Example length of data array

        message.setMessage(type, data, length);
        
        // Print out the message
        System.out.println("Type: " + message.getType());
        System.out.print("Data: ");
        for (byte b : message.getData()) {
            System.out.print(Integer.toHexString(b & 0xFF) + " ");
        }
        System.out.println("\nLength: " + message.getLength());
    }
}
```

Sample: 13 MidiFileFormat_4
----------------------------------------------------------------
`int getByteLength()`: This method returns the length in bytes of the entire MIDI file's data, including the header chunk and all the track chunks.

Code snippet:
```java
import javax.sound.midi.MidiSystem;
import javax.sound.midi.MidiFileFormat;
import javax.sound.midi.InvalidMidiDataException;
import java.io.File;
import java.io.IOException;

public class Main {
    public static void main(String[] args) {
        File midiFile = new File("path/to/your/midi/file.mid");
        
        try {
            MidiFileFormat midiFileFormat = MidiSystem.getMidiFileFormat(midiFile);
            int byteLength = midiFileFormat.getByteLength();
            System.out.println("Byte length of the MIDI file: " + byteLength + " bytes");
        } catch (InvalidMidiDataException | IOException e) {
            e.printStackTrace();
        }
    }
}
```

Make sure to replace `"path/to/your/midi/file.mid"` with the actual path to your MIDI file.
