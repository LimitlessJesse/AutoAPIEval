Sample: 0 OutputKeys
----------------------------------------------------------------
1. `String get(String key)`: This method retrieves the value for the specified key in the output properties.
2. `void set(String key, String value)`: This method sets the output property for the specified key to the specified value.
3. `String remove(String key)`: This method removes the output property associated with the specified key.
4. `String getMethod()`: This method retrieves the method for the output format.
5. `void setMethod(String method)`: This method sets the method for the output format to the specified value.
================================================================

Sample: 1 Transformer
----------------------------------------------------------------
1. `void setOutputProperty(String name, String value)`: This method sets an output property that will be in effect for the transformation.
2. `void transform(Source xmlSource, Result outputTarget)`: This method transforms the specified XML Source and sends the transformation result to a Result object.
3. `void setParameter(String name, Object value)`: This method allows a parameter to be set for the transformation.
4. `void setURIResolver(URIResolver resolver)`: This method sets the URIResolver used by the transformer for resolving URIs in documents().
5. `void setOutputProperties(Properties oformat)`: This method sets the output properties for the transformation.

Sample: 2 TransformerFactory
----------------------------------------------------------------
1. Transformer newTransformer(): This method creates a new Transformer object that performs a series of transformations on a source XML document.
2. Templates newTemplates(Source source): This method creates a Templates object, which represents a compiled stylesheet that can be used to transform XML documents.
3. Transformer newTransformerHandler(): This method creates a ContentHandler which acts as a Transformer, allowing XML to be passed through to SAX ContentHandler.
4. Source getAssociatedStylesheet(Source source, String media, String title, String charset): This method retrieves the associated stylesheet of a source XML document based on the provided media, title, and charset.
5. void setURIResolver(URIResolver resolver): This method sets the URIResolver used for resolving URIs in the stylesheet.
